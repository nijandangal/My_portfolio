ESP32 LED Control System

In this workshop, you'll learn how to build a complete LED control system using the ESP32 microcontroller. This project covers hardware setup, firmware development, and web-based control interface.

What You'll Need

ESP32 Development Board

The main microcontroller unit. Features WiFi, Bluetooth, and multiple GPIO pins.

LED (x3)

Standard 5mm or 3mm LEDs. Use different colors (Red, Green, Blue) for visual feedback.

Pin Configuration

Component ESP32 Pin Function
Red LED GPIO 13 Digital Output (ON/OFF Control)
Green LED GPIO 12 Digital Output (ON/OFF Control)
Blue LED GPIO 14 Digital Output (ON/OFF Control)
LED Ground GND Common Ground

Connection Steps

  1. Connect the positive leg of each LED through a 220Î© resistor to the respective GPIO pin
  2. Connect the negative leg of each LED to the GND pin on ESP32
  3. Use breadboard for clean connections
  4. Verify all connections before powering up

Firmware Code

#include <WiFi.h>
#include <WebServer.h>

// WiFi credentials
const char* ssid = "YOUR_SSID";
const char* password = "YOUR_PASSWORD";

// Pin definitions
const int RED_LED = 13;
const int GREEN_LED = 12;
const int BLUE_LED = 14;

WebServer server(80);

void setup() {
  Serial.begin(115200);
  
  // Initialize LED pins
  pinMode(RED_LED, OUTPUT);
  pinMode(GREEN_LED, OUTPUT);
  pinMode(BLUE_LED, OUTPUT);
  
  // Turn off all LEDs initially
  digitalWrite(RED_LED, LOW);
  digitalWrite(GREEN_LED, LOW);
  digitalWrite(BLUE_LED, LOW);
  
  // Connect to WiFi
  WiFi.begin(ssid, password);
  while (WiFi.status() != WL_CONNECTED) {
    delay(1000);
    Serial.println("Connecting to WiFi...");
  }
  
  Serial.println("WiFi connected!");
  Serial.println(WiFi.localIP());
  
  // Setup web server routes
  server.on("/", handleRoot);
  server.on("/led", handleLED);
  server.on("/status", handleStatus);
  server.begin();
}

void loop() {
  server.handleClient();
}

void handleLED() {
  if (server.hasArg("color")) {
    String color = server.arg("color");
    
    if (color == "red") {
      digitalWrite(RED_LED, !digitalRead(RED_LED));
    } else if (color == "green") {
      digitalWrite(GREEN_LED, !digitalRead(GREEN_LED));
    } else if (color == "blue") {
      digitalWrite(BLUE_LED, !digitalRead(BLUE_LED));
    }
  }
  server.send(200, "application/json", "{\"status\":\"ok\"}");
}

void handleStatus() {
  String status = "{";
  status += "\"red\":" + String(digitalRead(RED_LED)) + ",";
  status += "\"green\":" + String(digitalRead(GREEN_LED)) + ",";
  status += "\"blue\":" + String(digitalRead(BLUE_LED));
  status += "}";
  server.send(200, "application/json", status);
}

Getting Started - Step by Step

1

Setup Hardware

Connect the LEDs and resistors to the ESP32 following the pin configuration table above. Use breadboard for clean connections.

2

Install Tools

Download Arduino IDE and install ESP32 board support package. This enables you to program the ESP32 board.

3

Upload Code

Copy the firmware code provided above into Arduino IDE. Update WiFi credentials and upload to your ESP32.

4

Configure WiFi

Replace YOUR_SSID and YOUR_PASSWORD with your actual WiFi credentials. Check Serial Monitor for connection status.

5

Access Dashboard

Open the IP address shown in Serial Monitor in your browser to see the LED control dashboard.

6

Control LEDs

Click buttons to turn LEDs on/off. Control from your phone or any device on the same WiFi network!

Troubleshooting Tips

LEDs Not Turning On

Check if positive and negative legs are connected correctly. LEDs are polarity-sensitive. Use a multimeter to verify connections.

WiFi Connection Issues

Verify SSID and password are correct. Check Serial Monitor output. Ensure your WiFi is on 2.4GHz (ESP32 doesn't support 5GHz).

Upload Errors

Select correct COM port and board "ESP32 Dev Module". Install drivers if port is not detected.

Can't Access Dashboard

Copy the IP address from Serial Monitor. Make sure your phone is on same WiFi network. Use http:// (not https://)
